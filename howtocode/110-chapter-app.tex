The GeoCache Calculator, the GCC, was developed by Daniel 'Eisbehr' Kern and Mark 'S-Man42' some years ago. But unfortunately the development stands still for more than six years. Several attempts to revive the project failed. So Mark decided to start a new project. So it is possible to add new features, fix bugs and maybe even approach the Apple world.



\section{The Application - some words about its structure}

The GCWizard provides its numerous tools with six menues:
\begin{description}
    \item[Coordinates] provides tools to calculate coordinates
    \item[Encryption and encodings] provides algorithms for common encoding/decoding and encryption/decryption operations
    \item[Formula solver] provides the ability to create complex formulas with variables and solve them
    \item[Games] provides algorithms for common encoding/decoding operations basing on games
    \item[Science and technology] provides algorithms for common encoding/decoding operations basing in the periodic table and other scientific stuff
    \item[Symbol tables] provides tables from a various number of languages and codings
\end{description}

This structure gives a first impression how the source code will be organized.
