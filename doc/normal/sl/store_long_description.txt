GC Wizard je odprtokodna zbirka geolovskih orodij.

Projekt se je začel kot geolovski pripomoček za reševanje ugank na terenu za delo brez podatkovne povezave. V GC Wizard so vključena številna orodja za preproste kriptografske rešitve ter geografske, matematične in druge izračune. Vključenih je tudi več sto različnih pisav.

Obseg projekta se je sčasoma močno povečal, zato je aplikacija uporabna tudi za druge, negeolovske namene.

<b>Highlights</b>

<b>Splošna orodja</b>
• Formula Solver: For handling multi stage variables
• Multi Decoder: Enter an unknown code and let several decoders and calculators interpret it in a row
• More than 200 sets of Symbol Tables: Direct decoding symbols to characters; save own encoding as an image
• Online Manual: Every tool has its own manual page, translated into 🇬🇧 🇩🇪 🇫🇷

<b>Kriptografija in kodiranje</b>
• Alphabet Values (A = 1, B = 2, ...): Configurable alphabets with language specific special character handling
• Braille graphical decoder: Type the points into a graphical interface; supports different languages
• Book Cipher: Choose the correct system (e.g. Line + Letter Number or Section + Line + Word Number, ...), handle special characters and empty lines, ...
• Enigma: A full working Enigma simulator incl. numerous possible settings
• Esoteric Languages: Generators and interpretors for several esoteric programming languages like Brainf**k, Ook, Malbolge and Chef
• Morse
• Numeral Words: Lists of important numbers in different languages. For English + German there are parsers to identify even complex numeral words
• Substitution and Vigenère Code Breakers: Try to find the solution without knowing the keys
• Classic Codes: Playfair, Polybios, Railfence, ...
• Historic codes: Caesar, Vigenère, telegraph codes, ...
• Military codes: ADFGX, Cipher Wheel, Tapir, ...
• Technical encodings: BCD, CCITT, Hashes (incl. brute-force hash breaker), RSA, ...

<b>Koordinate</b>
• Preračunavanje koordinat z algoritmi, ki omogočajo zelo natančne izračune tudi za zelo velike razdalje in vedno upoštevajo obliko Zemlje (elipsoid)
• Podpora za različne elipsoide, tudi za druge planete
• Format koordinat: podpora za UTM, MGRS, XYZ, SwissGrid, NAC, PlusCode, Geohash, ...
• Projekcija točk, vključno z natančno obratno projekcijo
• Open Map: Set own points and lines, measure paths, export as and import from GPX/KML; OpenStreetMap and satellite view
• Variable Coordinate: Interpolate coordinate formulas if some parts of coordinate are not given. Prikaz rezultatov na zemljevidu
• Cross bearing, Center point of two and three coordinates, different intersections of lines and circles, ...

<b>Znanost in tehnologija</b>
• Astronomija: izračun položaja sonca in lune na določeni lokaciji ob določenem času
• Color Space Converter: Convert color values between RGB, HSL, Hex, CMYK, ...
• Countries: ISO, calling and vehicle registration codes, flags
• Date and Time functions: Weekday, time differences, ...
• Irrational Numbers: π, φ and e: Show and search up to > 1 Mio. digits
• Number sequences: Factorial, Fibonacci and Co.
• Numeral Systems: Converts decimal to binary, hexadecimal, ...
• Periodic Table of the Elements: Interactive view; lists which order the elements by any criterion
• Phone Keys: Converts classic phone keys to letters. Supports phone model specific behaviours
• Prime Numbers: Search prime numbers up to 1 Mio.
• Segment Display: Graphical interface for decoding and encoding 7 to 16 segment displays
• Unit Converter: Length, Volume, Pressure, Power and many more; convert between common units incl. prefixes like micro and kilo
• Apparent Temperature, Cross Sums, DTMF, Keyboard layouts, Projectiles, Resistor codes, ...

<b>Slike in datoteke</b>
• Hex Viewer
• Exif/Metadata Viewer
• Analyse frames of Animated Images
• Color corrections: Adjust contrast, saturation, edge detection, ...
• Search for Hidden Data or hidden archives
• Read QR/Barcodes from images, create them from binary input
